About the Project

This Project is Jacob Yealy's capstone project for CSC-499 at Shippensburg University.
In this section, you can find the experiment design information, project components, and results.

Purpose

The purpose of this project is to analyze the impact that sentiment has on the United States Stock Market.
To do so, two LSTM networks were trained. Both networks were trained on an 80/20 training and testing split.
One network only utilized stock data, while the second utilized both stock and sentiment data.
Using the resulting predictions from the models, a positive or negative impact of sentiment on stock price could be analyzed.

Dependencies

This project utilizes the following libraries and APIs: TensorFlow - Machine Learning Keras - Machine Learning SQLite - Database YFinance - Stock data source Alpha Vantage - Sentimental data source Flask - Fullstack connection Chart.js - The chart object
